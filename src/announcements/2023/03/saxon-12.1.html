Announcing Saxon 12.1!

The Saxon 12.1 maintenance release has been published. This is a maintenance release for Java, C#, C, PHP, and Python that fixes a number of issues reported since the first Saxon 12.0 release.

Saxon 12.1 was released on 21 March 2023. This release has been uploaded to the usual locations on the Saxonica website, GitHub, and Maven. SaxonCS 12.1 is built with .NET 6.

For a list of the issues resolved in this release, please visit the issue tracker in the Saxon project on our support site.

Download products:

  • Binaries for PE and EE are available from our download page.
  • SaxonJ-HE is available on the Maven Central repository.
  • SaxonJ-HE, PE, and EE can also be found on our experimental Maven repository: https://dev.saxonica.com/maven/.
  • Saxon-HE is no longer distributed on SourceForge. It is now available from our open source GitHub repository. The GitHub repository also provides source code for those who need it.

For more details, please consult the documentation.
